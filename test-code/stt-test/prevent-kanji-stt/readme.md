According to the post [faster-whisperで漢字を抑制してひらがな、カタカナ出力する #Python - Qiita](https://qiita.com/maishikawa/items/dcadfeebf693080f0415) , whisper is able to transcribe kana only, without kanji outputing.

If you want to run the script here, please download the `multilingual.tiktoken` file at [openai/whisper - Github](https://github.com/openai/whisper/blob/main/whisper/assets/multilingual.tiktoken) and save it at the same directory of this readme file.
